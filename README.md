# AI Teaching App

An AI-powered teaching assistant that helps students learn through clear explanations, adaptive questioning, and interactive feedback.

This project is developed using an **Agile workflow**, with documented planning, sprints, and retrospectives.

---

## Purpose
- Support student learning across a wide range of subjects
- Adapt explanations based on individual understanding
- Encourage comprehension over memorization

---

## Development Process
- Agile / sprint-based development
- User stories and backlog-driven features
- Iterative improvement through reflection and testing

Project documentation is maintained in the `docs/` directory.
